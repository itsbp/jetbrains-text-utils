<!-- Plugin description -->
Text Utils is a collection of simple text utilities

#### Features
* Sort lines in selection in Ascending Order
* Sort lines in selection in Descendign Order
* De-duplicate lines in selection
* Convert selected text to Uppercase
* Convert selected text to Lowercase
* Base64 Encode selected text
* Base64 Decode selected text

## Installation

- Using IDE built-in plugin system:
  
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "Text Utils"</kbd> >
  <kbd>Install Plugin</kbd>
  
- Manually:

  Download the [latest release](https://github.com/itsbp/jetbrains-text-utils/releases/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
